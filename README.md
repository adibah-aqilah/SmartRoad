# SmartRoad - Road Hazard Reporting App

SmartRoad is a premium, community-driven Android application designed to improve road safety by allowing users to report and track road hazards in real-time. Built with a focus on accessibility and high-contrast UI, the app ensures that every user can contribute to a safer driving environment.

## 🚀 Key Features

*   **Real-Time Reporting:** Quickly report potholes, flooding, fallen trees, and other hazards with photos and descriptions.
*   **GPS Precision:** Accurate location tagging using Google Maps and Fused Location services, optimised for physical hardware including POCO/MIUI devices.
*   **Interactive Hazard Map:** View nearby reports on a live map with custom status markers.
*   **Status Tracking:** Monitor the progress of reports from 'New' to 'Investigating' and 'Resolved'.
*   **Accessibility First:** High-contrast colour schemes and custom theme handling to bypass aggressive system dark modes (MIUI Force Dark).
*   **Secure Authentication:** User accounts managed via Firebase Authentication.

## 🛠️ Tech Stack

*   **Language:** Java / Android SDK
*   **Backend:** Firebase Firestore (Database) & Firebase Auth
*   **Maps:** Google Maps SDK & Google Play Services Location
*   **Image Handling:** Glide (for Base64 and URL image processing)
*   **UI/UX:** Material Design Components with custom accessibility optimisations

## 📋 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/adibah-aqilah/SmartRoad.git
   ```
2. Open the project in **Android Studio**.
3. Ensure you have a valid `google-services.json` file in the `app/` directory for Firebase services.
4. Build and run the app on an Android device or emulator.
